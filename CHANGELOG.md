## [0.2.0] - 2026-02-12

### Features

- Add comprehensive task management with GTD-style support
- Tasks support 5 statuses: todo, inprogress, blocked, done, cancelled
- Tasks support 4 priorities: low, medium, high, urgent
- Tasks can link to ideas and projects
- Due date support for tasks
- Full task lifecycle: new, list, show, status, priority, due, link, unlink, edit, delete
